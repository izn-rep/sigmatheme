Sigma App v1.0

Responsive Web App Kit

created: Nov 17, 2016

by: LeThemes

web: lethemes.net

email: lethemes@gmail.com

Overview

Sigma is a Clean and simple responsive admin dashboard template built with bootstrap 3.3.7 and HTML5, CSS3, jQuery. Sigma very easy customization and lightweight which is basically designed for the developers with powerful layouts. It follows the best practices using the latest technologies and components such as LESS, Jade, Gulp, Browser Sync, Bower and Git.

Features

  • Clean and simple
  • Powerful layouts
  • Valid HTML5 and CSS3
  • Built with bootstrap 3.3.7
  • Uses LESS, Jade, Gulp, Browser Sync, Bower, Git
  • Easy to understand HTML Structure, CSS Classes & Javascript markup
  • Fully responsive design, supported for desktops, laptops, smartphones and tablets
  • And much more…

Install

  1. Install a local server
  2. Install Node.js
  3. Install Git
  4. Install Bower and Gulp - Run npm install -g bower gulp-cli
  5. Install node modules - Run npm install
  6. Run gulp to start server

Note: Run Cmd as Administrator in Windows

I have defined all the task by default. Gulp will compile less, js and jade files. You can modify or add your task in gulpfile.js

Commands list:

  • gulp: start server
  • gulp jade: complied jade file and beautify them
  • gulp less: complied less file and minify them
  • gulp js: minify js file
